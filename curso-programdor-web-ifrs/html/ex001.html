É o cabeçaho da página, a Tag "header".
É o menu da página, a Tag "nav".
É a seção da página, exemplo: Noítcias, Esportes, Lazer, a Tag "section".
É o rodapé da página, a Tag "footer".

A quebra de linha pode ser feita pela tag "br", que não precia ser fechada.

Também podemos quebra um texto por parágrafo com a tag "p".

Para inserir linha horizontal, utlizar a tag "hr".

Para comentários no código fonte deve-se usara a tag
